# What you learned
- I learned what it means to build a website for someone other than yourself, which can be difficult depending on the client's ability to communicate what theywant out of the project and the vision for the final project they have in mind.
- I had a lot of practice with aligning classes in the html files to classes in the css file. Moving forward, I will try to use more specific class names to make it easier to find the exact class in the css file I am looking for while making style changes.
- I learned that the longer a file gets, the more difficult it is to stay organized. For this reason, it is hepful to add /* */ where possible to avoid confusion
